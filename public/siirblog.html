🔗 www.senolsoyleyici.com.tr

Bu proje benim için sadece bir yazılım değil, bir kültür aktarımıydı. Hem sevdiklerimin mirasını yaşatmak, hem de full-stack development yolculuğumda ciddi bir adım atmaktı.

🛠️ Kullandığım Teknolojiler

  • React 18 + TypeScript
  • Tailwind CSS ile sade, mobil uyumlu tasarım
  • React Router DOM ile SPA deneyimi
  • Node.js + Express.js ile RESTful API altyapısı
  • MongoDB Atlas
  • JWT tabanlı authentication sistemi
  • Vercel ile hem frontend hem backend deploy

💡 Proje Özellikleri

  • Admin paneli üzerinden şiir ekleme, düzenleme, silme
  • Şiir arama, detay ve listesi sayfaları
  • Mobil uyumlu sade ve hızlı kullanıcı deneyimi
  • Şiirlere özel görüntülenme sayıları
  • Yorum onaylama/reddetme
